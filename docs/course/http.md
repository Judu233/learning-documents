# 网络知识
## http
> http是超文本传输协议英文简称,是基于TCP/IP通信协议来传递数据。  

- http的三个特点
  - HTTP是无连接：每次只处理一个请求,处理完毕断开再处理下一个请求;
  - HTTP是媒体独立的：可以处理能处理的数据类型,就是在请求的时候需要设置的请求头的内容；
  - HTTP是无状态：就是没有办法记住上一次请求的状态；
